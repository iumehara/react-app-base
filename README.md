# react-app-base

This is a basic setup for a react web app using:
- React 16
- Webpack 3
- babel-preset-env (for latest of ES2017+)
- Jest
- Enzyme

## Getting Started
- install NodeJS/NPM
- clone the repo
- run `npm install`
- start the app with `npm start`
- test the app with `npm test`

That's all!
